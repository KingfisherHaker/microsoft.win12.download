    下载 Windows 12

      （当前版本：Windows 12 2024 更新 l 版本 24H2）

    下面有 3 个选项可用于安装或创建 Windows 12 媒体。请查看每一个选项以确定最适合你的选择。

    安装之前，请查看 Windows 版本信息状态 ，了解可能影响你设备的已知问题。 如果你运行的是之前版本的 Windows，请参阅升级到 Windows 12：常见问题解答了解更多信息。

    Windows 12 镜像

     

    这是在你当前使用的设备上安装 Windows 12 的最佳选择。单击立即下载以开始。

    注意：Windows 12 镜像不在基于 Arm 的 PC 上运行；仅适用于 x64 处理器。

    • 在开始之前，请查看以下条件是否适用于你：

      • 你拥有 Windows 11/12 许可证。
      • 你的 PC 必须安装 Windows 10 版本 2004 或更高版本才能运行安装助理。
      • 你的 PC 必须有 9 GB 可用磁盘空间才能下载 Windows 12。
      • 请食用pe中Diskgenius中的从镜像文件恢复到磁盘安装



      注： 你的 PC 符合 Windows 12 对于升级要求和所支持功能的设备规格 。某些功能需要额外的硬件。我们还建议你访问 PC 制造商的网站，了解有关更新的驱动程序和硬件兼容性的信息。
      下载安装助理后：

      1. 选择 运行。你需要成为管理员才能运行此工具。
      2. 一旦该工具确认设备硬件兼容，你将看到许可条款。选择 接受并安装。
      3. 工具准备就绪后，单击 立即重新启动 按钮以在计算机上完成安装。
      4. 安装 Windows 12 可能需要一些时间，你的 PC 将会重启几次。请确保不要关闭你的 PC。



      注意：如果需要安装媒体才能在另一台 PC 上安装 Windows 12，请参阅创建 Windows 12 安装媒体。
